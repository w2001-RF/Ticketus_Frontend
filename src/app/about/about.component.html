About Ticketus

Ticketus is a web application built with a robust .NET 8 backend and a modern TypeScript-based frontend (Angular/React). It allows users to efficiently manage tickets through a CRUD interface. Users can create, update, view, and delete tickets within a seamless and user-friendly interface.

Key Features:

  • Create, read, update, and delete tickets
  • Intuitive UI with tables to display ticket data
  • Supports pagination, sorting, and filtering for large datasets
